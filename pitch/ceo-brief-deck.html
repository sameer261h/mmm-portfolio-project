The ethos

We name exactly what each part is allowed to do.

No exaggerated "autonomous AI" claims — every component is labeled at the maturity level it actually operates at.

Planner
L1
one structured call drafts a paused campaign — no autonomy claimed
Analyst
L2
chooses its own read-only tools to answer a question
Operator
L3
proposes a change — a human approves every single one

The honest gap: our tests check that the code works. We don't yet score whether the AI's judgment was actually good — that's the next investment, named here rather than hidden.

04 / 05
The moat

The moat isn't a clever prompt.

Anyone can wire an LLM to an ad API and call it autonomous. The rare, defensible part is knowing exactly which decision belongs to statistics, which to the model's reasoning, and which must stay with a person — then actually shipping that discipline instead of the flashy version.
