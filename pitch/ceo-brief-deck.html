The product, not a diagram

Five screenshots, one running app.

Captured live from the local build — not mockups. The same demo is running at the link on the last slide.

Phase 1 — Planner. MMM budget split (56% Search / 44% PMax) becomes a rationale and an editable, paused-by-default campaign draft.
Phase 2 — Analyst. Asked “why did CPA rise?” — the agent chooses its own read-only tools (get_performance, get_search_terms) and answers from real numbers.
Phase 3 — Operator. “Cut the search budget” becomes a structured ticket with a reason and impact estimate — Approve/Reject sits right there, nothing applies itself.
Phase 5 — Simulated optimizer loop. The agent checks the account on its own initiative, catches a 6.8x CPA spike, and proposes a pause — same human gate, still nothing auto-applies.
Phase 6 — Meta Ads. The MMM's social channel recommendation (+75%, the largest reallocation found) becomes a paused Meta Feed campaign, live-verified against a real ad account.
04 / 06
The ethos

We name exactly what each part is allowed to do.

No exaggerated "autonomous AI" claims — every component is labeled at the maturity level it actually operates at.

Planner
L1
one structured call drafts a paused campaign — no autonomy claimed
Analyst
L2
chooses its own read-only tools to answer a question
Operator
L3
proposes a change — a human approves every single one

We built a judgment eval, not just code tests — 14 scripted scenarios with known-correct answers, scoring whether the AI's proposed action was actually right. Result: a deterministic reference gets every one right, every run; the LLM path scores a 33% restraint rate and is only fully consistent on 5 of 14. We report that finding rather than hide it — it's exactly the kind of gap code tests alone can't catch.

05 / 06
The moat

The moat isn't a clever prompt.

Anyone can wire an LLM to an ad API and call it autonomous. The rare, defensible part is knowing exactly which decision belongs to statistics, which to the model's reasoning, and which must stay with a person — then actually shipping that discipline instead of the flashy version.

06 / 06
What's next

Three moves in progress.

1. Autonomous initiation, planned but not built: the agent starts its own monitoring runs and earns permission to auto-apply exactly one low-risk, reversible action — never a budget or pause decision — only once the eval suite proves it's reliable enough, checked fresh on every run.
2. Meta Ads final write step, pending a payment method that clears region/currency checks.
3. A feedback loop that checks whether each applied change actually worked, and feeds that outcome back into the next decision.
